LANGKAH-LANGKAH MENGGUNAKAN APLIKASI PENCATATAN BARANG :
1. Aktifkan Localhost XAMPP MySQL
2. Buat database yang ada di file Query MySQL Pencatatan Barang.sql
3. Jalankan Aplikasi Pencatatan Barang.jar
4. Setelah dijalankan :
- fungsi button "SIMPAN" untuk menyimpan data
- fungsi  button "CARI" untuk mencari kode barang dengan memasukan terlebih dahulu ID'nya di kolom ID
- fungsi button "HAPUS" untuk menghapus data dengan memasukan terlebih dahulu ID'nya di kolom ID
fungsi button "EDIT" untuk mengubah data'nya dengan memasukan terlebih dahulu ID'nya di kolom ID kemudian langsung masukan data lainnya di kolom lainnya, dengan patokan ID'nya
- fungsi button "BATAL" untuk membatalkan penambahan data
- fungsi button "KELUAR" untuk keluar dari aplikasi'nya

catatan : jika aplikasi tidak jalan, mungkin bisa dari JDK'nya, developer membuatnya dengan menggunakan JDK 8
